Modelo de Aprendizagem Linear Voted Perceptron¶

Este python notebook descreve a implementao do modelo de aprendizagem Voted Perceptron que uma variao do Perceptron, que um modelo de aprendizagem linear ideal para problemas linearmente separveis.

Perceptron¶

O Perceptron um modelo de aprendizagem linear. Seu funcionamento se baseia no mtodo do gradiente descendente para encontrar um hiperplano que divide o conjunto de dados linearmente, por esta razo, o perceptron indicado para problemas linearmente separveis.

Dataset¶

Neste trabalho utilizaremos o conjunto de dados Breast Cancer Wisconsin (Diagnostic) Data Set. Este conjunto de dados contm dados computados a partir de imagens de tumores de mama e uma label definindo se o tumor maligno ou bengno.

Voted Perceptron¶

Agora podemos implementar a variante do modelo Perceptron chamada Voted Perceptron.

A proposta deste modelo melhorar ter um melhor desempenho em conjuntos de dados linearmente separveis com grandes margens de separao. No algoritmo, so armazenadas mais informaes durante o treinamento para utilizar estas informaes e, assim, gerar resultados melhores nas iteraes seguintes do treinamento.

class VotedPerceptron:
    def __init__(self, n_iter):
        self.n_iter = n_iter
        self.V = []
        self.C = []
        self.k = 0
    
    def fit(self, x, y):
        for i in range(len(y)):
            if y[i] == 0:
                y[i] = -1
        k = 0
        v = [np.ones_like(x)[0]]
        c = [0]
        for epoch in range(self.n_iter): # runs through the data n_iter times
            for i in range(len(x)):
                pred = 1 if np.dot(v[k], x[i]) > 0 else -1 # checks the sing of v*k
                if pred == y[i]: # checks if the prediction matches the real Y
                    c[k] += 1 # increments c
                else:
                    v.append(np.add(v[k], np.dot(y[i], x[i])))
                    c.append(1)
                    k += 1
        self.V = v
        self.C = c
        self.k = k

    def predict(self, X):
        preds = []
        for x in X:
            s = 0
            for w,c in zip(self.V,self.C):
                s = s + c*np.sign(np.dot(w,x))
            preds.append(np.sign(1 if s>= 0 else 0))
        return preds
